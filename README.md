# AviUtl2用カラーパレット（アイドルマスター）

## 概要

ブランドごとに、ブランドカラーや属性色、アイドル個人カラーをカラーパレットとして利用できるようにまとめたものです。

## 使い方

1. `.palette` ファイルを `C:\ProgramData\aviutl2\Default`の中にコピーする
2. AviUtl2を起動し、色設定パネルの🔁ボタンからパレットを選択する
  - ![パレット選択](img/1.png)
3. 色を選択する
  - ![色選択](img/2.png)
